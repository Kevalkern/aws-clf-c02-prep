# Glossary

## Availability Zone

An isolated location within an AWS Region. Multiple Availability Zones are used to improve resilience and availability.

## CAPEX

Capital expenditure. Large upfront spending on infrastructure.

## CloudTrail

AWS service that records API activity for auditing and governance.

## CloudWatch

AWS service for metrics, logs, alarms, and operational monitoring.

## Consolidated Billing

Billing feature used with AWS Organizations to combine charges across accounts.

## DynamoDB

Managed serverless NoSQL database service.

## EBS

Block storage designed for use with Amazon EC2.

## EC2

Elastic Compute Cloud. Virtual servers in AWS.

## Edge Location

Point of presence used by services such as CloudFront to deliver content closer to users.

## EFS

Managed elastic file storage that can be mounted by multiple compute resources.

## Elasticity

Ability to rapidly scale resources up or down based on demand.

## GuardDuty

Managed threat detection service for AWS accounts and workloads.

## IAM

Identity and Access Management. Controls authentication and authorization in AWS.

## IAM Identity Center

Service for centralized workforce access to AWS accounts and business applications.

## KMS

Key Management Service. Used to create and manage encryption keys.

## Lambda

Serverless compute service that runs code in response to events.

## Least Privilege

Security principle of granting only the permissions required to perform a task.

## MFA

Multi-factor authentication. Adds a second verification factor for sign-in.

## OPEX

Operational expenditure. Paying for usage as an ongoing operating cost.

## Region

Physical geographic area containing multiple Availability Zones.

## Reserved Instances

Pricing model for committing to usage over time in exchange for lower cost for certain services.

## RDS

Managed relational database service.

## Role

An AWS identity with permissions that can be assumed temporarily by users, applications, or AWS services.

## Route 53

AWS DNS and traffic routing service.

## S3

Scalable object storage service.

## Savings Plans

Commitment-based pricing model that can reduce compute costs with more flexibility than some reserved pricing models.

## Security Group

Stateful virtual firewall associated with resources in a VPC.

## Shared Responsibility Model

Model defining what AWS secures and what the customer secures.

## Shield

Managed DDoS protection service.

## SNS

Simple Notification Service. Pub/sub messaging and notifications.

## SQS

Simple Queue Service. Managed message queue for decoupling systems.

## Spot Instances

Discounted compute pricing for interruptible workloads.

## Trusted Advisor

Service that provides recommendations for cost optimization, security, limits, and performance.

## VPC

Virtual Private Cloud. Isolated network environment in AWS.

## WAF

Web Application Firewall. Helps protect web applications from common web exploits.
