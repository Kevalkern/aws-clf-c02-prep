# Flashcards

Use these as quick active-recall drills. Read the front first, answer out loud, then check the back.

## Card 1

Front: What is the difference between `CAPEX` and `OPEX` in cloud economics?

Back: `CAPEX` is upfront infrastructure spending. `OPEX` is pay-as-you-go operational spending.

## Card 2

Front: What is an `AWS Region`?

Back: A physical geographic area that contains multiple Availability Zones.

## Card 3

Front: What is an `Availability Zone`?

Back: An isolated location within a Region, used to improve resilience and availability.

## Card 4

Front: What is an `edge location` mainly used for?

Back: Delivering content and services closer to users, especially with CloudFront.

## Card 5

Front: Shared responsibility model: what does AWS secure?

Back: Security `of` the cloud, such as facilities, hardware, and foundational managed infrastructure.

## Card 6

Front: Shared responsibility model: what does the customer secure?

Back: Security `in` the cloud, such as IAM, data, configurations, and guest OS patching on EC2.
