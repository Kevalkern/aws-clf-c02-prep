# High-Yield Checklist

Use this as the minimum bar before sitting the exam.

## Domain 1: Cloud Concepts

- I can explain Regions, Availability Zones, and edge locations.
- I can explain elasticity, scalability, agility, and fault tolerance.
- I can explain CAPEX vs OPEX.
- I can explain the AWS shared responsibility model.
- I can name the Well-Architected Framework pillars.

## Domain 2: Security And Compliance

- I know IAM users, groups, roles, and policies.
- I know why the root user should be avoided for daily work.
- I know MFA and least privilege.
- I know IAM Identity Center at a high level.
- I know AWS Organizations and multi-account basics.
- I know KMS, Secrets Manager, ACM, Shield, and WAF.
- I know CloudTrail, Config, GuardDuty, and Security Hub.
- I know the difference between AWS security of the cloud and customer security in the cloud.

## Domain 3: Cloud Technology And Services

- I know when to use EC2, Lambda, ECS, EKS, and Fargate at a high level.
- I know the basic purpose of S3, EBS, EFS, FSx, and S3 Glacier.
- I know the basic purpose of RDS, Aurora, DynamoDB, and ElastiCache.
- I know the basic purpose of VPC, Route 53, CloudFront, API Gateway, VPN, and Direct Connect.
- I know the basic purpose of SQS, SNS, EventBridge, and Step Functions.
- I know CloudWatch vs CloudTrail vs Config.
- I know migration basics for DMS, Migration Hub, and Snow Family.

## Domain 4: Billing, Pricing, And Support

- I know On-Demand vs Reserved vs Spot vs Savings Plans.
- I know AWS Free Tier at a high level.
- I know Budgets, Cost Explorer, and Cost and Usage Reports.
- I know consolidated billing and Organizations basics.
- I know the major differences between AWS support plans.

## Exam Behavior

- I can eliminate obviously wrong services quickly.
- I can identify the lowest-ops managed option in a scenario.
- I can spot when the question is really about cost, security, or availability.
- I can handle multi-response questions without over-selecting.

## Red Flags

If any of these are weak, fix them before exam day:

- confusing CloudWatch with CloudTrail
- confusing IAM roles with IAM users
- confusing S3, EBS, and EFS
- weak understanding of pricing models
- weak understanding of shared responsibility
