# Service Map

This is a study map, not the full official service list. It focuses on the services most likely to matter in CLF-C02 prep.

Official in-scope reference:

- `https://docs.aws.amazon.com/aws-certification/latest/cloud-practitioner-02/clf-02-in-scope-services.html`

## Compute

- `Amazon EC2`: virtual servers
- `AWS Lambda`: serverless functions
- `Elastic Load Balancing`: distribute traffic
- `AWS Auto Scaling`: adjust capacity automatically
- `Amazon Lightsail`: simplified VPS-style deployments
- `AWS Elastic Beanstalk`: app deployment platform

## Storage

- `Amazon S3`: object storage
- `Amazon EBS`: block storage for EC2
- `Amazon EFS`: shared file storage
- `Amazon FSx`: managed file systems
- `Amazon S3 Glacier`: archival storage
- `AWS Backup`: centralized backup management

## Database

- `Amazon RDS`: managed relational database
- `Amazon Aurora`: AWS relational engine compatible with MySQL/PostgreSQL
- `Amazon DynamoDB`: serverless NoSQL key-value/document database
- `Amazon ElastiCache`: in-memory caching
- `Amazon DocumentDB`: document database

## Networking

- `Amazon VPC`: isolated networking
- `Amazon Route 53`: DNS and routing
- `Amazon CloudFront`: CDN
- `Amazon API Gateway`: managed API front door
- `AWS Direct Connect`: dedicated private connectivity
- `AWS VPN`: encrypted connectivity
- `AWS PrivateLink`: private service access

## Security

- `AWS IAM`: identity and access control
- `AWS IAM Identity Center`: workforce access across accounts and apps
- `AWS KMS`: encryption key management
- `AWS Secrets Manager`: secret storage and rotation
- `AWS WAF`: web application firewall
- `AWS Shield`: DDoS protection
- `AWS Artifact`: compliance reports
- `Amazon GuardDuty`: threat detection
- `AWS Security Hub`: security findings aggregation
- `AWS Config`: resource configuration tracking

## Monitoring And Governance

- `Amazon CloudWatch`: metrics, logs, alarms
- `AWS CloudTrail`: API activity and audit history
- `AWS Organizations`: multi-account management
- `AWS Control Tower`: landing zone governance
- `AWS Trusted Advisor`: cost, security, and performance recommendations
- `AWS Well-Architected Tool`: architecture review support

## Integration And Serverless

- `Amazon SQS`: message queue
- `Amazon SNS`: pub/sub notifications
- `Amazon EventBridge`: event routing
- `AWS Step Functions`: workflow orchestration
- `AWS Fargate`: serverless container compute

## Cost And Support

- `AWS Budgets`: spending thresholds and alerts
- `AWS Cost Explorer`: cost analysis
- `AWS Cost and Usage Reports`: detailed billing export
- `AWS Marketplace`: third-party software catalog
- `AWS Support`: support plans and cases

## Migration

- `AWS DMS`: database migration
- `AWS Migration Hub`: migration tracking
- `AWS Application Migration Service`: server migration
- `AWS Snow Family`: offline/edge data transfer

## Analytics And ML

- `Amazon Athena`: query data in S3
- `Amazon Redshift`: data warehouse
- `Amazon QuickSight`: BI dashboards
- `AWS Glue`: data integration and ETL
- `Amazon Kinesis`: streaming data
- `Amazon SageMaker AI`: ML platform
- `Amazon Rekognition`: image and video analysis
- `Amazon Textract`: document text extraction

## High-Value Comparisons

- `CloudWatch` vs `CloudTrail` vs `Config`
- `S3` vs `EBS` vs `EFS`
- `RDS` vs `DynamoDB`
- `EC2` vs `Lambda`
- `SNS` vs `SQS`
- `Shield` vs `WAF`
- `Direct Connect` vs `VPN`
